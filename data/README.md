# Sample Data

This folder contains sample data for testing the tools in this repository.

| Folder | Contents |
|---|---|
| `data_in_raw/` | Raw paired-end FASTQ reads, untrimmed |
| `data_in_trimmed/` | Paired-end FASTQ reads after adapter/quality trimming |
| `data_fasta/` | Assembled contigs/scaffolds in FASTA format |
| `data_fna/` | Reference and example genomic FASTA (`.fna`) files |
| `data_gff/` | Genome annotation files in GFF format |
